Sistema de Gestión Académica

Este proyecto consiste en el desarrollo de un sistema de gestión académica implementado en Python, que permite administrar información relacionada con usuarios, estudiantes, cursos, inscripciones y calificaciones. El sistema funciona a través de una interfaz de consola y aplica conceptos fundamentales de programación.

Descripción

El sistema está diseñado para simular el funcionamiento básico de una plataforma académica. Permite registrar usuarios con diferentes roles (estudiante o profesor), gestionar estudiantes y cursos, realizar inscripciones y registrar notas.

Toda la información se maneja en memoria utilizando estructuras de datos como listas y diccionarios, lo que permite ejecutar el sistema sin necesidad de una base de datos externa.

Funcionalidades
Registro e inicio de sesión de usuarios
Asignación de roles (estudiante y profesor)
Creación y listado de estudiantes
Creación y listado de cursos
Inscripción de estudiantes en cursos
Registro de notas
Consulta de notas e inscripciones
Visualización del estado académico
Generación de resumen académico
Requerimientos Funcionales

El sistema permite la gestión básica de la información académica mediante las siguientes funcionalidades:

Registro de usuarios con validación de duplicados
Inicio de sesión con credenciales
Registro y consulta de estudiantes
Creación y visualización de cursos
Inscripción de estudiantes con validación de cupos
Registro de notas en un rango válido (0.0 a 5.0)
Consulta de información académica
Generación de reportes básicos
Tecnologías utilizadas
Python
Programación estructurada
Listas y diccionarios
Estructura del proyecto

El proyecto está organizado en funciones que permiten separar cada una de las responsabilidades del sistema, como la gestión de usuarios, estudiantes, cursos e inscripciones.

Cómo ejecutar el proyecto
Descargar o clonar el repositorio:
git clone https://github.com/AndresMurcia23/GestionAcademica.git
Ingresar a la carpeta del proyecto:
cd GestionAcademica
Ejecutar el programa:
python main.py
