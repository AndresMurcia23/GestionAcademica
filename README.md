# Sistema de Gestión Académica

Este proyecto consiste en el desarrollo de un sistema de gestión académica implementado en Python, que permite administrar información relacionada con usuarios, estudiantes, cursos, inscripciones y calificaciones.

El sistema funciona a través de una interfaz de consola y aplica conceptos fundamentales de programación.

---

## Descripción

El sistema está diseñado para simular el funcionamiento básico de una plataforma académica. Permite:

- Registrar usuarios con diferentes roles (estudiante o profesor)
- Gestionar estudiantes y cursos
- Realizar inscripciones
- Registrar notas

Toda la información se maneja en memoria utilizando listas y diccionarios.

---

## Funcionalidades

- Registro e inicio de sesión de usuarios  
- Asignación de roles  
- Creación y listado de estudiantes  
- Creación y listado de cursos  
- Inscripción de estudiantes en cursos  
- Registro de notas  
- Consulta de notas e inscripciones  
- Visualización del estado académico  
- Generación de resumen académico  

---

## Requerimientos Funcionales

El sistema permite la gestión básica de la información académica:

- Registro de usuarios con validación de duplicados  
- Inicio de sesión con credenciales  
- Registro y consulta de estudiantes  
- Creación y visualización de cursos  
- Inscripción con validación de cupos  
- Registro de notas en rango de 0.0 a 5.0  
- Consulta de información académica  
- Generación de reportes básicos  

---

## Tecnologías Utilizadas

- Python  
- Programación estructurada  
- Listas y diccionarios  

---

## Estructura del Proyecto

El proyecto está organizado en funciones que separan responsabilidades como:

- Gestión de usuarios  
- Gestión de estudiantes  
- Gestión de cursos  
- Gestión de inscripciones
