# Código Fuente - Sistema de Gestión Académica

Esta rama contiene la implementación principal del sistema de gestión académica desarrollado en Python.

Aquí se encuentra el archivo `main.py`, donde está toda la lógica del programa organizada en funciones.

---

## Contenido de la rama

- `main.py`: archivo principal que contiene:
  - Gestión de usuarios (registro y login)
  - Gestión de estudiantes
  - Gestión de cursos
  - Inscripciones
  - Registro de notas
  - Menús interactivos según el rol

---

## Estructura del código

El sistema está desarrollado bajo un enfoque de programación estructurada, utilizando:

- Funciones para separar responsabilidades
- Listas como estructuras de almacenamiento
- Diccionarios para representar entidades (usuarios, estudiantes, cursos, etc.)
- Control de flujo mediante condicionales y bucles

---

## Conceptos implementados

- Estructuras de datos contenedoras
- Conversión de tipos de datos (casting)
- Validación de entrada de datos
- Manejo de excepciones (`try-except`)
- Búsqueda secuencial (complejidad O(n))
- Mutabilidad de estructuras de datos
